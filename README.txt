ІНСТРУКЦІЯ ДЛЯ ЗАПУСКУ БОТА НА RENDER (БЕЗКОШТОВНО)

1. Створи бота через BotFather у Telegram:
   - /start
   - /newbot
   - Введи назву та username (username закінчується на _bot)
   - Отримай токен (формат: 1234567890:ABCDefGhIjKlmNoPQRstuVWxyz)

2. Створи канал у Telegram:
   - Додай бота в адміністратори з правом постити повідомлення
   - Запам'ятай @username каналу

3. Відкрий файл bot.py:
   - Замініть ВСТАВ_СВІЙ_ТОКЕН на токен бота
   - Замініть @ВСТАВ_СВІЙ_КАНАЛ на @username каналу

4. Залий цей проект на GitHub:
   - Створи новий репозиторій
   - Завантаж усі файли (bot.py, requirements.txt, README.txt)

5. Перейди на https://render.com:
   - Створи новий Web Service
   - Підключи свій GitHub-репозиторій
   - У полі Command введи:
     python bot.py
   - Python версія: 3.10+

6. Запусти сервіс — бот почне автоматично:
   - Щогодини постити новини з Yahoo Finance (заголовок українською, посилання, картинка)
   - Щодня о 16:30 постити карту ринку з Finviz
